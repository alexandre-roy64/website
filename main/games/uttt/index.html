Ultra Tic-Tac-Toe





Ultra Tic-Tac-Toe is an enhanced version of Super Tic-Tac-Toe (also called Ultimate Tic-Tac-Toe) which is itself an enhanced version of the Tic-Tac-Toe we all know.

Features:

  • New customizable rules (comparing to Super Tic-Tac-Toe)
  • Single player mode with 2 difficulty levels for the opponent
  • Multiplayer mode (on the same device)
  • English and French languages support
  • A "fun" secret to unlock while beating the opponent in hard difficulty... 🤫

A few words from the developer:

This is my first time developing AND releasing a "full" game, so I am not expecting anyone to play or having fun with it, and it may still have some bugs that have not yet been found. So if you found one, contact me. And if you somehow liked the game, thank you very much!
